# Prompt Enhancer

## Описание

Это расширение для Google Chrome, которое улучшает ввод промпта с помощью AI. Он использует Google Gemini API для улучшения промптов. Расширение позволяет сохранять API ключ, выбирать режим улучшения (basic или detail) и копировать результат.

## Технологический стек

* HTML
* CSS
* JavaScript
* Google Chrome Extension API
* Google Gemini API

## Установка

1. Скачайте расширение с GitHub.
```bash
git clone https://github.com/UniQlow17/prompt-enhancer.git
```
2. Откройте расширение в Chrome [здесь](chrome://extensions/).
3. Нажмите на кнопку "Загрузить распакованное расширение".
4. Выберите файл, который вы скачали.
5. Расширение будет установлен.

## Использование

1. Откройте расширение в правом верхнем углу браузера.
2. Введите свой промпт в поле ввода.
3. Выберите режим улучшения (basic или detail).
4. Нажмите кнопку "Улучшить промпт".
5. Результат будет отображен ниже.
6. Чтобы скопировать результат, нажмите кнопку "Копировать".

## Настройки

### API ключ

1. Введите свой API ключ в поле ввода.
2. Нажмите кнопку "Сохранить и продолжить".

### Режим улучшения

1. Выберите соответствующую кнопку (basic или detail).
